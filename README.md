# Netflix Data Cleaning & Preprocessing Project  
### Data Analyst Internship — Elevate Lab

This project was completed as part of my Data Analyst Internship Task at Elevate Lab, demonstrating the full workflow of data cleaning, preprocessing, documentation, and reporting using Excel and Python. The goal was to transform a raw, inconsistent dataset into a clean, analysis-ready version following professional data quality standards.

## Data Cleaning Steps

1. Handling Missing Values (Contextual Imputation)  
2. Removing Duplicates  
3. Fixing Invalid Ratings (“74 min” → “Not Rated”)  
4. Converting Date Formats to DD-MM-YYYY  
5. Standardizing Text Values  
6. Standardizing Column Headers  
7. Validating Data Types
